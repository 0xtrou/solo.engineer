📋 Setup Guide

  1. Push your Next.js/React app to GitHub
  2. Connect repo to Vercel or Netlify — auto-deploys on push
  3. Register a domain on Porkbun, point DNS to hosting
  4. Enable Cloudflare proxy for DDoS protection + caching
  5. Add R2/B2 for user uploads and static assets
  6. Set up CI/CD with GitHub Actions for staging previews
