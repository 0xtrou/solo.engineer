1. AI SEO Content Agent (SeoBOT Clone)

A. What is it? An AI-powered SEO agent that autonomously researches keywords, generates optimized blog posts, manages programmatic SEO (pSEO) pages, and deploys mini-apps — all through a dashboard. SeoBOT is at $73,038 MRR with 7% growth, proving the market pays for hands-off SEO.

B. 30-day verdict: YES Core product is keyword research API + LLM content gen + CMS publish. All three are commoditized. Differentiation is in the autonomous loop (crawl → analyze → write → publish → monitor).

C. Tech Stack

  • Frontend: Next.js 15 + Tailwind CSS + shadcn/ui
  • Backend: Node.js (Hono) on Fly.io
  • Database: Supabase (Postgres + pgvector for content similarity)
  • Infra: Cloudflare Workers for edge pSEO pages
  • AI APIs: OpenAI GPT-4o-mini ($0.15/1M input) for content, Claude Haiku for analysis

  • Week 2: Content generation engine (prompt templates, SEO scoring, internal linking). CMS integrations (WordPress REST API, Webhook for static).
  • Week 3: pSEO page generator (template system, bulk create, sitemap). Rank tracking via SerpAPI. Dashboard v1.
  • Week 4: Stripe billing. Email reports. Landing page. Beta launch on Product Hunt + Indie Hackers.

  3. SeoBOT has head start → Compete on price + specific vertical (e.g., e-commerce SEO only)
