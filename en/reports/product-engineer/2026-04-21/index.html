1. DM Champ — White-Label AI Sales Agent

A. What is it?

An AI-powered sales agent that handles outbound prospecting, lead qualification, and follow-up conversations via chat/email. White-labeled so agencies resell it under their brand. Currently at $191K MRR with 4% growth on TrustMRR.

B. 30-Day Verdict: YES

Core product is a conversational AI wrapper with CRM integration. The white-label layer is a config-driven skin. Doable in 30 days for an MVP.

C. Tech Stack

  • Frontend: Next.js 15 + Tailwind CSS (dashboard, agent config UI)
  • Backend: Node.js/Express or FastAPI (Python for LLM orchestration)
  • Database: PostgreSQL (conversations, leads) + Redis (session state)
  • Infra: Docker on Hetzner/Vultr ($24/mo), Cloudflare CDN
  • AI APIs: GLM-4 ($0.5/M input, $0.5/M output) for conversations, GPT-4o-mini ($0.15/M input) for lead scoring

D. OSS Dependencies

Name Purpose License
LangChain LLM orchestration, tool chains MIT
Vapi / Retell Voice agent SDK (optional v2) MIT
NextAuth Multi-tenant auth MIT
Resend Transactional email MIT

E. 30-Day Sprint Plan

  • Week 1: Core chat engine + GLM API integration + basic lead schema
  • Week 2: Dashboard UI (agent config, conversation logs), CRM webhook integrations (HubSpot, Close)
  • Week 3: White-label system (custom domain, branding, reseller portal), email outreach channel
  • Week 4: Billing (Stripe), polish, onboarding flow, launch

F. Monthly Costs at 1K Users

Item Cost
AI API (GLM-4, ~500 conversations/day × 2K tokens avg) ~$300
Infra (2× Hetzner CX32) $48
Email (Resend 50K/mo) $20
Total ~$370/mo

G. Revenue Model

  • Starter: $49/mo (1 agent, 500 convos/mo)
  • Pro: $149/mo (5 agents, CRM integrations)
  • Agency: $399/mo (white-label, unlimited agents)
  • Conversion rate: 8% free→paid, time to $1K MRR: ~6 weeks

H. Top 3 Risks + Mitigation

  1. Competition is fierce (DM Champ, Artisan, Clay) → Differentiate on vertical: focus on real estate or dental first
  2. LLM hallucination in sales → Hard guardrails + human-in-loop for first outreach
  3. White-label complexity → Ship single-tenant first, add multi-tenant in v2

I. Verdict: BUILD

Proven model ($191K MRR exists), clear monetization, technically straightforward AI wrapper.

2. Postiz — Agentic Social Media Scheduler

A. What is it?

An AI-powered social media management tool that doesn't just schedule posts — it generates content, suggests optimal timing, and autonomously engages with audiences. $97K MRR, 18% growth. Open-source core with hosted SaaS.

B. 30-Day Verdict: PARTIAL

Core scheduling + AI content gen is doable in 30 days. The "agentic engagement" (auto-reply, auto-DM) needs more polish and safety rails.

  • Backend: Node.js + BullMQ (job queue for scheduling)
  • Database: PostgreSQL + Redis (queue state)
  • Infra: Railway or Fly.io ($20-50/mo starter)
  • AI APIs: GLM-4 for content generation ($0.5/M tokens), Claude Haiku for engagement replies ($0.25/M input)

  • Week 2: AI content generation (GLM-4), template system, image gen (DALL-E mini)
  • Week 3: Analytics dashboard, optimal timing engine, multi-account support
  • Week 4: Billing, onboarding, engagement automation (auto-like, auto-reply) — basic only

5. Bookedin — No-Code AI Receptionist Builder

A. What is it?

A platform that lets businesses create AI-powered phone receptionists without code. Handles appointment booking, FAQs, call routing. Currently at $51K MRR on TrustMRR and listed FOR SALE — indicating the model works but founders exited.

B. 30-Day Verdict: YES

Core is: voice AI + calendar integration + FAQ knowledge base. All solvable with existing APIs. The no-code builder is a form-driven config UI.

C. Tech Stack

  • Frontend: Next.js + Tailwind (builder UI, dashboard)
  • Backend: Node.js/Express (telephony webhook handler)
  • Database: PostgreSQL (businesses, agents, call logs)
  • Infra: Hetzner CX32 + Twilio/Vonage for telephony
  • AI APIs: GLM-4 ($0.5/M) for conversation, Whisper ($0.006/min) for STT, TTS ($0.015/min)

D. OSS Dependencies

Name Purpose License

E. 30-Day Sprint Plan

  • Week 1: Voice pipeline (Twilio → Whisper → GLM-4 → TTS), basic call handling
  • Week 2: Knowledge base builder (upload FAQ docs, auto-index), calendar integration (Google Calendar)
  • Week 3: No-code builder UI (configure agent personality, business hours, routing rules)
  • Week 4: Dashboard (call logs, analytics), Stripe billing, onboarding wizard, launch

F. Monthly Costs at 1K Users

Item Cost
AI API (STT + LLM + TTS, ~200 calls/day × 3 min avg) ~$400
Telephony (Twilio, ~60K min/mo) ~$360
Infra (Hetzner + Redis) $40
Total ~$800/mo

G. Revenue Model

  • Starter: $49/mo (1 agent, 200 min/mo)
  • Business: $149/mo (3 agents, 1000 min/mo, analytics)
  • Enterprise: $399/mo (unlimited, CRM integrations)
  • Conversion: 10%, time to $1K MRR: ~4 weeks (fewer users needed at higher price)

H. Top 3 Risks + Mitigation

  1. High per-user costs (telephony + voice AI adds up) → Prepaid minute bundles, overage charges
